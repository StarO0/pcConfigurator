# PC Configurator Frontend v6.0

Next.js 16 interface for the configurator, catalogue, comparison, accounts and data admin.

The Data workspace exposes:

- ready-product coverage;
- Open Icecat + Polish shop acquisition batches;
- staging and manual review;
- CSV/XLSX/JSON import;
- source configuration and URL queue;
- PostgreSQL backup/restore.

Frontend lint and production build are executed during the Docker image build.
