# Простой запуск PC Configurator

## Первый запуск

1. Установи **Docker Desktop**.
2. Запусти Docker Desktop и дождись надписи **Engine running**.
3. Распакуй архив в обычную папку, например `C:\Projects\pc-configurator`.
4. Дважды нажми **`START_PROJECT.bat`**.
5. Первый запуск скачивает образы и собирает проект. После готовности браузер откроется автоматически.

Адреса:

- сайт: `http://localhost:3000`
- Swagger API: `http://localhost:8000/docs`
- локальный администратор: `admin@pcbuilder.app`
- пароль: `Local-admin-123`

## Остановка

Запусти **`STOP_PROJECT.bat`**.

## Когда старые контейнеры или база мешают запуску

Запусти **`RESET_AND_START.bat`**. Он удалит только локальные Docker-данные этого проекта и соберёт всё заново.

## Когда запуск снова завершился ошибкой

Файл **`STARTUP_ERROR_LOG.txt`** создаётся автоматически рядом со скриптом. В нём уже находятся статусы контейнеров и логи backend, frontend и PostgreSQL.
