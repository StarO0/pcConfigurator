PC CONFIGURATOR - КАК ЗАПУСТИТЬ

1. Установи и запусти Docker Desktop.
2. Дождись надписи Engine running.
3. Дважды нажми START_PROJECT.bat.
4. Сайт откроется по адресу http://localhost:3000

Логин администратора:
admin@pcbuilder.app
Local-admin-123

Если старый запуск мешает: RESET_AND_START.bat
Остановить проект: STOP_PROJECT.bat
При ошибке смотри STARTUP_ERROR_LOG.txt
