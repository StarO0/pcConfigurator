# Итог реализации v6.1

## Имеем сейчас

- 332 демонстрационных товара с ценой PLN, фото, URL предложения и наличием;
- PostgreSQL 15 как единственную runtime-базу, постоянный Docker volume и Alembic;
- автоматическую очистку legacy-карточек без фото или активного предложения;
- Open Icecat lookup без приватного ключа по GTIN или бренд+MPN;
- canonical provenance, Icecat ID, главное фото, галерею и характеристики;
- польские price collectors для Morele, Komputronik, RTV EURO и x-kom;
- строгий quality gate и variant-safe дедупликацию;
- несколько offers на один продукт, историю цен, stale status и price alerts;
- рабочий конфигуратор, совместимость, сравнение, ручную сборку, сохранение и аккаунты;
- PostgreSQL custom backup/restore;
- исправленные Windows launchers без кириллических CMD-команд;
- безопасный XML preview/import, API сравнения 2–4 товаров и матрица сравнения во frontend;
- интерфейсы интеграции AI с детерминированной проверкой результата.

## Проверено в этой сборке

- backend: 42 passed;
- Ruff check и format check: passed;
- Open Icecat real public endpoint: получены Icecat ID, фото, галерея и характеристики;
- fresh starter seed: ровно 332 активных товара, 332 offers, 100% фото;
- повторный seed: остаётся 332 товара, дублей не создаёт;
- PostgreSQL Alembic offline generation: head `77e6121a6d20`;
- frontend ESLint: passed;
- frontend Next.js production build: passed.

В среде сборки не был установлен Docker daemon, поэтому контейнерный runtime-тест здесь не
запускался. Он включён в `test-project.sh` и `test-project-windows.bat` и выполняет реальный
smoke test на PostgreSQL у пользователя.

## Чего всё ещё нет

- полноценного нормализованного каталога: starter snapshot не содержит EAN/MPN и имеет неполные specs;
- финальной модели/промптов/оценки качества AI;
- юридически подтверждённых соглашений или официальных feeds польских магазинов;
- гарантии, что магазин не изменит HTML/anti-bot правила;
- полного Icecat/FULL Icecat bulk dump без регистрации;
- собственной CDN-копии изображений;
- production SMTP, домена, TLS, мониторинга и внешнего backup storage;
- реальных benchmark-лицензий для всех новых моделей.

Эти пункты не мешают локальному запуску и наполнению публично доступными товарами, но нужны
до коммерческого production-запуска.
