# PC Catalog Collector

Расширение без API-ключей читает только открытую пользователем карточку товара. Оно извлекает
JSON-LD, Open Graph, цену, фотографию и таблицу характеристик, а затем сохраняет JSON.

1. Распакуйте папку.
2. Откройте `chrome://extensions` или `edge://extensions`.
3. Включите режим разработчика и нажмите «Загрузить распакованное расширение».
4. Выберите эту папку, откройте карточку товара и нажмите значок расширения.
5. Полученный JSON загрузите в «Данные → Catalog Harvester» проекта.

Расширение не обходит сайт автоматически, не требует логина и не отправляет данные наружу.
